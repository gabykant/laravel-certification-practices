# Laravel Certification – Practice Kit (100 Questions)

This repository helps PHP developers prepare seriously for the Laravel certification exam.

It focuses on:

- real framework knowledge
- architecture thinking
- production patterns
- not memorization

## How it works

1. Open `questions.md`
2. Answer all questions (A/B/C/D)
3. Write your answers on paper or notes
4. Open `answers.md`
5. Compare and score yourself

## Scoring

90–100 → Ready for the exam  
75–89 → Good level, review weak areas  
60–74 → More practice needed  
< 60 → Study fundamentals again

## Coverage

Questions are based on:

- routing & middleware
- service container & providers
- request lifecycle
- queues, cache, events
- security
- database & eloquent
- testing
- performance & architecture

## Tips

- understand "why", not just "what"
- practice with a real project
- know queues, caching and testing well
- be comfortable with internals

Good luck and feel free to contribute improvements.
